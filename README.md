# QR Code Generator

Проект реализует генерацию QR-кодов на основе входного реестра с возможностью
кастомизации внешнего вида и экспорта результата.

## Статус проекта

На текущий момент реализована основная функциональность в рамках требований,
зафиксированных на момент начала и в процессе разработки.


## Реализовано

- генерация QR-кодов на основе входных данных;
- поддержка кастомизации внешнего вида;
- формирование и обработка реестра;
- предпросмотр результата;
- подготовка к массовой генерации и экспорту.
